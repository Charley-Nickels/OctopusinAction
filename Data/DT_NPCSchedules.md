# DT_NPCSchedules

Data table using `STR_NPCSchedule` that seeds example NPC routines.

- Rows include 3 example NPCs (Clerk, Technician, Resident) with simple home/work vectors such as `(-300,0,0)` homes and `(300,0,0)` work points, and work hours like 9–17 or 10–18.
- Located at `/Data/DT_NPCSchedules` for the NPC manager to load at runtime.
