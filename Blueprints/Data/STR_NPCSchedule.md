# STR_NPCSchedule

Blueprint structure describing a simple NPC daily schedule.

- Fields:
  - `NPCName` (Name)
  - `HomeLocation` (Vector)
  - `WorkLocation` (Vector)
  - `StartWorkHour` (Integer)
  - `EndWorkHour` (Integer)
- Stored at `/Blueprints/Data/STR_NPCSchedule` for use by the NPC schedule data table and runtime manager.
